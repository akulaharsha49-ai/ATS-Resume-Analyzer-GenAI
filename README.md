# ATS Resume Analyzer (GenAI)

This project analyzes resumes against job descriptions using Generative AI
and provides ATS compatibility score and improvement suggestions.

## Features
- Resume analysis
- Job description matching
- ATS compatibility score
- Improvement suggestions

## Technologies Used
- Python
- Streamlit
- Google Gemini API

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Add your API key in .env file
GOOGLE_API_KEY=your_api_key

3. Run the application
streamlit run app.py
